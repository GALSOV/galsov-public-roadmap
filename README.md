# GALSOV — Public Roadmap & Project Board

GALSOV (Galactic Sovereignty) is a deterministic, turn-based 4X space strategy game being built as a learn-by-building public experiment.

This repository is **roadmap-only**:
- ✅ Roadmap / milestones / Kanban board
- ✅ Design notes that are safe to share
- ✅ Devlog links, screenshots, progress updates
> [!IMPORTANT]
> Early milestones are foundation-heavy (determinism, TurnTrace, turn resolution). Expect engineering progress first; visuals come later.
- ❌ No engine source code
- ❌ No production game assets
- ❌ No build pipeline secrets / keys / internal tooling

## Where is the code?
The engine repository remains **private**.

Public progress is tracked here through milestones, issues, project board movement, screenshots, devlog links, and other public-safe notes.

## Current status
- Current public focus: v0.2 — Turn Machine v1 + Proof & Reporting Foundation
- Determinism-first: same seed -> same run -> same digest / trace expectations
- UI remains a reporting lens over the sim, not the source of truth
